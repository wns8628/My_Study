조건문

 * prompt('입력해라');      : 입력할수있는창이뜬다. (문자로받는다  1234 입력해도 '1234' 이걸로처리) 

 조건문은 if로 시작한다. if 뒤의 괄호에 조건이 오고, 조건이 될 수 있는 값는 Boolean이다.
 Boolean의 값이 true라면 조건이 담겨진 괄호 다음의 중괄호 구문이 실행된다.

	if(true 나 false){ 코드 }


        else       : 처음 if문이 false면 이걸로 다른조건확인 

        else if    : 조건을여러개로나눌때

 	
        * if안에는 if가 또 나올수가있다. (다중if)


 *논리연산자 

  &&         : and연산자 즉 둘다만족해야 true
  ||         : or 연산자 하나만 만족해도 true


  ex)       id = prompt('아이디를 입력해주세요.');

		alert('인증 했습니다.');
			} else {
				    alert('인증에 실패 했습니다.');
				}
  
  이런식으로 쓸수있다 

  *  !               : !true 이렇게쓰면 false가됨 불린값을 역전시킨다.

 자바스크립트에서 false로 간주되는것 

   - NaN       : 0/0 같은 ? 
   - ''        : 빈문자열
   - undefined : 정의가안됨
   - a         : 값이할당안됨변수(초기화안된변수:undefined)
   - null      : 널값도 false
