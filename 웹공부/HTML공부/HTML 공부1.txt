1] 태그 문법 = 태그는 총 140개? 정도있는데 다안외워도됨 찾아가면서 하던지 중요한것만좀하면되겠지
 
 <!DOCTYPE html>           =    독타입태그 = 작성할태그가 어떤버전(어떠한표준을따르는 html 로작성됐는가를 
                                선언해줌(브라우저에게 알려줌) 현재는 html5 인 독타입을쓴다  

  
 <html><html>              =    제일큰태그이다 <head>태그와 <body>태그를 이태그로 감싸준다 


 <head></head>             =    본문이아닌태그는 이태그안에 넣는다.
 <title></title>           =    페이지 대표 이름바꾸는것(탭의제목)  
 <meta charset="urf-8">    =    글씨깨짐방지태그

 
 <body><body>              =    본문은 이태그안에다 넣는다.
 <h1></h1>                 =    대제목(h1=헤딩원의줄임말) 굵고큰문자로꾸며주고 + 줄바꿈
 <h2></h2>                 =    소제목 <h6> 까지있는듯.. 
 
 <li></li>                 =    리스트태그 단독으로 거의사용안됨 <ul>,<ol>태그로보통묶음
 
 <ul></ul>                 =    그룹핑 = 같은성격의 리스트들을 이태그로 묶어준다(unordered list)  
 <ol></ol>                 =    그룹핑 = 순서를 매긴다 1.2.3 이렇게(ordered list)
  
 <strong></strong>         =    감싸준 글씨강조(진하게)
 
 <p></p>                   =    단락을 표현할때 쓴다.  패러그래프(단락)의 줄임말 
                           =    더떨어트리거나 이렇게할려면 css를 써서할수있고 <br>태그를써서 줄바꿈을더하던지..

 <br>                      =    줄바꿈태그(p태그는 줄바꿈간격이 고정) 단락을표현할거면p태그가 더좋다

 
 
 
2]속성 문법  

 ex)href, target등 순서상관 x
 
 <a href ="주소"></a>                             =  링크를거는 태그 브라우저에게 링크라는걸알려줌
 <a href ="주소" target="_blank"></a> 	          =  새로운탭으로 열리게하고싶으면 
 <a href ="주소" title="툴팁적기"></a>            =  링크위에 커서대면 툴팁나오도록

 <img src="peko.png">                             =  이미지태그 속성 폴더안에넣어서 저렇게쓰던가 url쓰던가
 <img src="peko.png" alt="이미지정보이름">        =  alt 속성은 이미지가깨졌을때 나오는 텍스트를 나타냄 꼭이걸써라 
 <img src="peko.png" width="200" height="300">    =  높이,폭 속성을줘서 그림크기맞추기 두개다주면왜곡되니 한개만주는걸추천
 <img src="peko.png" title="툴팁임">              =  이것도마찬가지 툴팁나오도록속성 



3]표(table) 태그 = 웹페이지에구조화된정보를 표현, 예전에 레이아웃잡을때(홈피디자인) 많이쓰기도했다 

 
 <th></th>                                        =  각각의 열을표현할떄는 이걸로묶어주자(필드)
 <td></td>                                        =  테이블에들어갈 각각정보를 td로 감싸준다 테이블데이터(td) 
 <td rowspan="2"></td>                            =  row(행)을병합하는 속성! 병합할곳은 없애줘라 
 <td colspan="3"></td>                            =  컬럼(열)을 병합 병합할곳은 없애줘라
 <tr></tr>                                        =  table row 행을묶는태그 행을묶는다

 <thead><thead>                                   =  필드는 이걸로묶어주자
 <tfoot><tfoot>                                   =  맨밑에것 (합계 이런것)은 이렇게묶으면됨

 <table></table>                                  =  위에태그들을 테이블태그로묶어줌 
 <table border="1"></table>                       =  보더속성을주면 줄이생겨서 테이블같긴하다 근데안이뻐서 잘안씀 
                                                  =  걍꾸밀떄는 css를 이용해서 하는게더좋을듯     
 
 
 1.표를 table태그로 묶고
 2.행을 tr 태그로 묶고            
 3.각각을 td 태그로 묶으면되겠찌
