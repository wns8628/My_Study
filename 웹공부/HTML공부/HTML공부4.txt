3.인풋타입 제출방식 (사용자의 입력을 강제,유도 할수있다)
  
    -HTML5부터 input type값이 대폭 추가됨.
     이것들의 전반적인 특징은 사용자 입력값을 받을 때 좀더 강한 규제, 강제, 유도를 할 수 있다는 것과
     모바일에서 사용자들이 좀 더 값에 맞게 입력하기가 쉬워졌다는 것. 
 

 	추가된 인풋타입들 

	color  (색깔지정)
	
	date (년월일 까지입력)
	
	datetime     (국제표준시간의방식으로 서버에전송? 아직지원x)
	
	datetime-local  (현재 위치 표준시간전송 아직지원x)
	
	email (이메일형식만입력)
	
	month (년과 월까지입력)
	
	number (숫자만입력)
	
	range  (슬라이더생김 슬라이더에따라 값을전송)
	
	search (검색할때사용하는것이다 라는걸 명시 정보로서의가치를 높인다)
	
	tel    (전화번호를 입력)
	
	time   (년도 시간입력)
	
	url   (url형식입력)
	
	week (몇년도 몇번째주 인지 입력) 



  <input type="number"> 			 // 숫자만입력가능
  <input type="number" min="10" max="15">  	 // 숫자만입력가능 ,최소값10 최댓값15로 규제:규칙안맞으면 제출불가임

 이정도 만하고 나머지 부분은 따로필요할때보면될듯
