
*상속 (inherited)

  엘리먼트를 디자인할때 그 태그에 속해있는 하위의 태그가 같이 적용이됨
	
 	ex) html { color:red; }     = html는 부모니깐 자식태그 모두가다 레드가됨 .. 

	*모든 속성이 상속되는게아님 되는게있고 안되는게 있다 필요할때찾으면됨ex)border는 상속안됨 *


*캐스캐이딩(cascading) = 웹브라우저 사용자 저자 가 조화를이루는것

 우선순위 : 웹브라우저 < 사용자 < 저자 

 하나의 태그에 css가중첩되면 

 우선순위 : 태그  < class < id < style    (정교함이 높으면 우선순위가높다)
 
 * !important 를적으면 무조건 우선순위가 1등이됨
