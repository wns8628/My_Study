* font-size 
   
         px  =크기가 딱고정
         
         em  =가변적
        
         rem =가변적 -이게 승리자    (사용자를존중함)  -html태그에 비례한다! 


* color
 
      color name    = 컬러이름이 있어서 그냥그걸쓰면됨             color:red;
       
      hex           = 16진수로 컬러를 정함 정교하게                color: #FF0000;          
 
      rgb           = 모든컬러에 이름을못붙이니 정교하게색정할때   color: rgb(255,255,255);


* text-align (정렬)
 
      left          = 왼쪽정렬
      right         = 오른쪽정렬
      center        = 가운대정렬
 
      justify       = 양쪽골고루정렬..


*font

  - font-family : 원하는폰트1, 원하는폰트2, 원하는폰트3; 이런식으로쓰면됨 1폰트가없으면2가나오고 이거임
 ex)font-family :   arial,verdana,"helvetica Neue", serif;       =폰트장식있는 폰트중하나
                                                    sans-serif;  = 장식안함 폰트중 하나 
                                                    monospace;   = 고정간격폰트
                                                   여러가지가 많다..

  - font-weigt : bold;           =폰트 두께 굵게
   
  - line-height: 2;              =행간(줄간격 1.2가 기본)


 font : 위에것들 font에 한번에 모아서 쓸수있다.
